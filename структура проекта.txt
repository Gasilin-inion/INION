Проект: Двухступенчатый конвертер для конвертации страниц html в таблицу Excel, а зетем, после ручного редактирования таблицы, в библиотечный формат ИРБИС 64.

Структура:
e-Library_to_IRBIS/
├── files_to_pricess/
│   ├── file_1.html
│   ├── file_2.html
│   ├── file_3.html
│   └── file_4.html
├── files_to_edit/
│   └── table_of_articles.xlsx
├── files_to_imoprt_to_IRBIS/
│   ├── list_of_documents.txt
│   └── wrong_categories.txt
├── elibrary_excel.py
├── excel_irbis.py
├── ya_trans.py
├── ya_GPT.py
├── correction_functions.py
├── add_categories.py
├── edd_keywords.py
├── multifile_import.py
├── journal_list.json
├── purple.json
├── velvet_cat.json
├── persons.json
└── descriptors.json

Описание:
- `elibrary_excel.py`: Конвертер страниц из html в excel.
- `excel_irbis.py`: Конвертер таблиц из excel в формат ИРБИС 64.
- `ya_trans.py`: Модуль для перевода иноязычных аннотаций на русский язык.
- `ya_GPT.py`: Модуль для оптимизации аннотаций на основе YandexGPT.
- `correction_functions.py`: Модуль для автоматической корректуры заголовков.
- `add_categories.py`: Модуль для автоматического проставления тематических рубрик.
- `edd_keywords.py`: Модуль для автоматического проставления ключевых слов.
- `multifile_import.py`: Модуль для множественной загрузки файлов html.
- `journal_list.json`: База данных журналов.
- `purple.json`: IAM-токен для доступа к сервисам Yandex Cloud.
- `velvet_cat.json`: Рубрикатор.
- `persons.json`: Список нормализованных персоналий.
- `descriptors.json`: Список нормализованных ключевых слов.
- `journal_list.json`: База данных журналов.
- `files_to_pricess/`: папка для исходных html страниц.
- `files_to_edit/`: папка для таблиц excel.
- `files_to_imoprt_to_IRBIS/`: папка для исходных конечных файлов, предназначенных для импорта в ИРБИС 64.

Стек: Python 3,8.
