# Bibliographic converters
### Набор библиографических инструментов для работы с ИРБИС 64
### Автор: Андрей Гасилин

#### Пакет включает в себя:
+ Конвертер e-Library-Excel, предназначенный для конвертации библиографических метаданных из формата .html в табличный формат .xlsx для последующего редактирования
+ Конвертер Excel-IRBIS, предназначенный для конвертации готовых таблиц в формат ИРБИС 64 с последующим импортом в соответствующую библиотечную систему
+ Экстрактер ненормализованных ключевых слов с добавлением в таблицы автозамены
+ Конвертер таблиц автозамены в задания глобальной корректировки ИРБИС 64
  
#### Вспомогательные модули
+ Модуль пакетной загрузки файлов (multifile_import.py)
+ Модуль генерации ключевых слов (add_keywords.py) на основе заглавия, аннотации и авторских ключевых слов
+ Модуль автоматической корректировки заглавий (correction_functions.py)
+ Модуль проверки корректности рубрик (check_functions.py)
+ Модуль генерации рубрик (add_categories.py) на основе ключевых слов
+ Модуль автоматического перевода (ya_trans.py)
+ Модуль обработки метаданных с помощью YandexGPT (ya_GPT.py)
+ Модуль конвертации данных из формата .xlsx в формат .json (xlsx_to_json_converter.py)

#### Авторитетные файлы
+ Таблица перевода символов в верхний регистр в именах собственных для функции автоматической корректировки (a_facet.xlsx)
+ Нормализованный список рубрик (categories.txt)
+ Нормализованный список дескрипторов с вариантами написания в виде словаря (descriptors.json)
+ Нормализованный список персоналий с вариантами написания в виде словаря (persons.json)
+ Список выходных данных избранных научных журналов (journal_list.json)
